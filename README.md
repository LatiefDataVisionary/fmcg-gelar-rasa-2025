# fmcg-gelar-rasa-2025
Repositori ini berisi analisis data dan pengembangan model prediktif untuk Data Science Competition Gelar Rasa 2025 yang diselenggarakan oleh HIMASADA UPNVJT. Proyek ini berfokus pada analisis dataset sintetis transaksi penjualan produk Personal Care untuk mengidentifikasi inovasi, memprediksi tren, dan menganalisis kanibalisasi produk.
